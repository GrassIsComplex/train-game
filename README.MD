# Train Game

Simple game about changing the track diraction to route the trains to the correct destination.
All the code is in a single cpp file.

## Features

- Simple textures
- Pause (press 'p')
- Game over screen with restart option
- Lose condition (when three trains are sent to the wrong destination, number of mistakes are marked with red X's under the score indicator)
- Difficulty increase with score (Every 5 points the train speed increases)
- Highscore written to a file
- Sound effect when switching tracks
